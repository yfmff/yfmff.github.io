C++ CallOnce

发表于： 2022-06-07 分类于： cpp
字数：230 阅读：≈ 1分钟 浏览：

在某些特定情况下，某些函数只能在多线程环境下调用一次,就可以使用std::call_once()来保证函数在多线程环境下只能被调用一次。使用 call_once()的时候，需要一个once_flag作为 call_once() 的传入参数。

1// 定义于头文件 <mutex>
2template< class Callable, class... Args >
3void call_once( std::once_flag& flag, Callable&& f, Args&&... args );

使用实例：

 1#include <iostream>
 2#include <thread>
 3#include <mutex>
 4using namespace std;
 5
 6once_flag g_flag;
 7void do_once(int a, string b)
 8{
 9    cout << "name: " << b << ", age: " << a << endl;
10}
11
12void do_something(int age, string name)
13{
14    static int num = 1;
15    call_once(g_flag, do_once, 19, "luffy");
16    cout << "do_something() function num = " << num++ << endl;
17}
18
19int main()
20{
21    thread t1(do_something, 20, "ace");
22    thread t2(do_something, 20, "sabo");
23    thread t3(do_something, 19, "luffy");
24    t1.join();
25    t2.join();
26    t3.join();
27
28    return 0;
29}
